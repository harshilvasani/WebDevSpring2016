  • Hand Gesture Recognition System to Control Television

    The whole and sole purpose of this system is to ease the controlling of television with the use of gestures instead of hardware remote control. With the help of hand gestures already programmed in the system, one can perform all the functionalities that a remote control can perform, just with the help of one’s hands. For this purpose, our system should achieve the following objectives:
    1) Encode the gestures with functionality.
    2) Fetch the gestures performed by the user.
    3) Process the gestures and perform the functionality.

  • Two Player Wifi Card Game

    Its a two player Card game developed in C# in Visual Studios.
